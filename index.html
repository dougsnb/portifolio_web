Últimos Projetos

Controle de Abastecimento de Diesel

Aplicativo de controle de abastecimento de diesel foi desenvolvido para otimizar a gestão de combustível em operações logísticas. Com uma interface intuitiva, o sistema permite o cálculo automático do saldo do tanque após o abastecimento, além de sugerir ajustes na quantidade com base na capacidade do tanque. O aplicativo também registra um histórico detalhado de todas as transações de abastecimento, garantindo uma análise precisa do consumo ao longo do tempo

Portifólio Pessoal

Portifólio foi desenvolvido utilizando HTML e CSS, com foco em um design responsivo e moderno. A estrutura foi organizada com flexbox e grid para garantir uma disposição harmônica dos elementos. As imagens e textos são centralizados, proporcionando uma navegação intuitiva. Estilizações, como gradientes de fundo e sombras, foram aplicadas para melhorar a estética visual. Efeitos de transição foram implementados para interações mais suaves. A paleta de cores e a tipografia foram escolhidas para refletir minha identidade visual. O projeto também foi testado em diferentes dispositivos para assegurar uma experiência consistente. Este portfólio é um reflexo das minhas habilidades em desenvolvimento web e design.
